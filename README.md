# 秦皇岛中药学招聘每日简报

手机可安装的 PWA，聚合秦皇岛全行业招聘信息，只保留专业要求明确包含“中药学”的招聘公告。覆盖医院、疗养康养机构、政府事业单位、药企、药店药房、学校科研等用人单位。

## 本机运行

```powershell
npm install
npm run refresh
npm start
```

电脑打开 `http://localhost:4173`。手机与电脑连接同一 Wi-Fi 后，打开 `http://电脑局域网IP:4173`，再选择“添加到主屏幕”。

## 每日更新

运行 `install-automation.ps1` 会创建两个 Windows 计划任务：每天 07:30 更新数据，以及登录 Windows 后自动启动页面服务。电脑需要开机联网；错过更新时间后会在下次可用时运行。

## 匹配规则

- `strict`：正文或 Excel 岗位附件明确出现“中药学”。
- `review`：写有药学类或相关专业，同时提到中药岗位，需要人工核对。
- 排除成绩、体检、面试、拟聘公示等非新招聘公告。
- 每条记录保留匹配原文片段和原公告链接，最终以招聘单位公告为准。

当前已接入市卫健委、市人社局、第一医院、第二医院、妇幼保健院、九龙山医院、工人医院，以及搜狗微信公众号文章搜索和公开网页招聘平台搜索。同一标题会自动去重，优先展示官方来源。

## 公网部署

仓库内的 GitHub Actions 每天北京时间 07:30 自动抓取并部署 GitHub Pages。公网版本不依赖个人电脑开机，手机和电脑可在不同网络独立访问。
